# code-editor
A simple web app project of a code editor running on a browser. Based on a FreeCodeCamp Course!

Check it out [here!](https://youtu.be/Vs4hQsDpxww?si=8V_Z4G1aa8JcdVJm)


Features:
- Create, edit, and run code for HTML, CSS, and JS
- Write down tasks and assignments for the current project
- Create test cases for the project
- Runs a preview of the current project either in the same page or on a separate tab
- Save and Load the current project as a JSON file
- Display and Clear log results when running project
